# Locklane Scripts

Utility scripts for local development and CI smoke validation.

## Planned Scripts

1. `bootstrap-dev.sh` - validate local prerequisites and setup.
2. `run-plugin-tests.sh` - run plugin-side tests.
3. `run-resolver-tests.sh` - run resolver-side tests.
4. `smoke-e2e.sh` - minimal end-to-end flow on fixture project.

## Usage

```bash
./scripts/bootstrap-dev.sh
./scripts/run-resolver-tests.sh
./scripts/run-plugin-tests.sh
./scripts/smoke-e2e.sh
```
