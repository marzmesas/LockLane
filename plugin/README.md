# Locklane Plugin Module

PyCharm/IntelliJ Platform plugin implementation (Kotlin).

## Planned Responsibilities

1. Tool window and action wiring.
2. Planning orchestration and background task execution.
3. Resolver process management.
4. Patch preview/apply flow.
5. Verification reporting UI.

## Bootstrap Commands

```bash
cd plugin
./gradlew test
./gradlew runIde
```
